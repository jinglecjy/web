村野之夜

顾城 浓厚的黑夜， 把天地粘合在一起。 星星混着烛火， 银河连着水渠， 我们小小的茅屋， 成了月宫的邻居。 去喝一杯桂花茶吧！ 顺便问问户口问题。

冬天的河流

顾城 松疏的沙滩上， 横躺着上百只大木船； 它们像是疲乏了， 露出宽厚的脊背， 晒着太阳…… 多么辽阔呵！ 没有人声。 河岸边， 开满了耀眼的冰花； 沙洲上， 布满了波浪留下的足迹， ——微细的纹路； 黄锈的铁锚斜躺着， 等待着春天的绿波。 冰冻的河是蓝色的； 无云的天是蓝色的； 多么单纯的颜色， 阳光润湿了大地的皮肤。 毡毯一样的沙滩 睡熟了； 它是美丽的， 却没有——一枝生命的花朵。

希望

顾城 海潮无休止地摇晃大陆 邀它一同把日月追逐 大陆在睡梦中透口粗气 火山的烟尘就把天空烧糊 海潮变成了惊慌的海啸 一直跑回大海的深处
